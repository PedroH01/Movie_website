# Star Filmes

# Sobre o projeto

https://starfilmes.netlify.app/

Star Filmes é uma aplicação web criada através de um tutorial do YouTube. 

Está aplicação contém um sidebar básico e uma opção de dark e light mode. 

# Tecnologias utilizadas
## Front end
- HTML 
- CSS
- JS
